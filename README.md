### Features

- Multi-Language (English/Turkish)
- Dark Mode Theme
- SEO Friendly Movie Urls (detail/{movie-name}/{i})
- Newly Added Movies and Top Movies on Home Page
- Simple Movie Search
- About Page
- Header and Footer Components
- Sass used for styles
